PHP Mail - mail()

Description & Uses of PHP mail()

The PHP mail() function is used to send emails directly from the PHP scripts.

Simple Example - PHP mail(String $to, String $subject, String $message, String $header);

Simple PHP Syntax

  1. <?php
  2. mail(String $to, String $subject, String $message, String $header);
  3. ?>

Parameter & Return Type

String $to Specifies the To Address. (*Required)
String $subject Specifies the To Subject. (*Required)
String $message Specifies the Body of an eMail. (*Required)
String $header Specifies the headers like From:, Cc:, Bcc: and Date:

What is PHP mail() ?

  • It's used to send an Instant Electronic Mail through PHP Script
  • It returns TRUE if Success otherwise FALSE.
  • Header Information atleast contain "From:" part and the additional informations like Carbon Copy (Cc:), Blind Carbon Copy (BCC), Date, if neccessary.
  • It is available in Three flavors
    • Simple Plain Text eMail
    • HTML embeded eMail
    • eMail with Attachment
  • Simple Plain Text eMail:
    • No HTML Tags are rendered
    • The default Content-Type for Plain mail() is "text/plain; charset=us-ascii"
  • HTML embeded eMail:
    • HTML Tags are rendered
    • Inline Style Sheet must be used
    • No <html>, </html>, <head>, </head>, <title>, </title> <doctype> are used for best practice.
    • No .css Files are used
    • No .js Files are used.
    • The default Content-Type for HTML mail() is "text/html; charset=ISO-8859-1"
  • eMail with an Attachment:
    • Attachment can be sent either Plain or HTML eMail
    • No external Reference files can be sent.(Ex: https://www.bbminfo.com/YGen.pdf)
    • Maximum File Size of an Attachment is 5 MB
    • The default Content-Types are
      1. The default Content-Type of a text message is "multipart/mixed"
      2. The default Content-Type of an attachment file is "application/octet-stream"

  1. <?php
  3. /* Simple HTML eMail with PDF Attachment */
  4. $from = "support@bbminfo.com";

  12. /* Header Information */
  13. $header = "From: " . strip_tags($from);
  15. /* Boundary Information */
  17. $semi_rand = md5(time());
  18. $mime_boundary = "==Multipart_Boundary_x{$semi_rand}x";
  20. /* Header Information */
  22. $header .= "\nMIME-Version: 1.0\n";
  23. $header .= "Content-Type: multipart/mixed;\n";
  24. $header .= " boundary=\"{$mime_boundary}\"";
  26. /* Inner Operation for Message Attachment */
  28. $msg_attachment = "This is a multi-part message in MIME format.\n\n";
  29. $msg_attachment .= "--{$mime_boundary}\n";
  30. $msg_attachment .= "Content-Type: text/html; charset=\"iso-8859-1\"\n";
  31. $msg_attachment .= "Content-Transfer-Encoding: 7bit\n\n" . $html_message . "\n\n";
  32. $msg_attachment .= "--{$mime_boundary}\n";
  34. /* Inner Operation for File Attachment */
  36. $file = fopen($file_path, "rb");
  37. $data = fread($file, filesize($file_path));
  38. fclose($file);
  40. $data = chunk_split(base64_encode($data));
  42. $file_attachment = "Content-Type: {\"application/octet-stream\"};\n";
  43. $file_attachment .= " name=\"$file_path\"\n";
  44. $file_attachment .= "Content-Disposition: attachment;\n";
  46. $file_attachment .= " filename=\"$file_path\"\n";
  48. $file_attachment .= "Content-Transfer-Encoding: base64\n\n" . $data . "\n\n";
  49. $file_attachment .= "--{$mime_boundary}\n";
  51. $message = $msg_attachment . $file_attachment;
  53. /* Triggering Mail Function (Sending Mail with an Image) */
  55. $flg = mail($to, $subject, $message, $header);
  57. if($flg == true) {
  59. echo("Your message has been sent.");
  61. } else {
  62. echo("Failed to sent.");
  63. }
  65. ?>
